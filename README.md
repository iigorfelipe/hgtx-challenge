## Installation

```bash
git clone git@github.com:iigorfelipe/hgtx-challenge.git
```

- ### Backend

```bash
cd hgtx-challenge/backend
```

```bash
npm install
```

## Local Usage

- ### Backend

```bash
cd hgtx-challenge/backend
```

```bash
npm run dev
```
